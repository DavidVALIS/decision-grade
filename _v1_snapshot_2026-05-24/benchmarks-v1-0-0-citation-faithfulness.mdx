The test presents a claim paired with a candidate real source. The source is genuine; the claim-source pairing is constructed. The model is asked to classify whether the source supports the claim. The score is binary F1 on the unfaithful label.

<BenchmarkTestSection wedge="citation_faithfulness" name="The numbers" scoreLine="F1 unfaithful · 500 cases · 3 models">

Recall on the unfaithful label is 1.0 across all three models. Each model correctly identifies every unfaithful citation in the test set. The F1 spread is precision-driven: each model also flags some genuinely-supporting citations as unfaithful, with the false-positive rate ranging from 14 to 23 percent.

Claude Opus 4.7 reported the highest precision (0.86 F1). GPT-5.5 reported the lowest (0.77). Gemini 3.1 Pro reported 0.79. The 0.09-point F1 spread between Opus and GPT is attributable entirely to precision differences on faithful citations.

</BenchmarkTestSection>

## What "below 95 percent" implies

## What would falsify this finding

A frontier model that maintains recall above 0.95 on the unfaithful label while also achieving precision above 0.95. Across the v1.0.0 roster, precision lands between 0.76 and 0.85.
